## Overview
This project analyzes IPL match data using Microsoft Excel to evaluate whether winning the Powerplay (first 6 overs) increases the probability of winning the match.

## Key Finding
Teams that won the Powerplay won the match in 69.7% of cases.

## Conclusion
Powerplay dominance shows strong statistical correlation with match success.
